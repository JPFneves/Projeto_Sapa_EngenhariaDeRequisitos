# Projeto_Sapa_EngenhariaDeRequisitos
O SAPA (Sistema de Automação de Presença Acadêmica) é um software em Python que automatiza a chamada universitária via leitura de código de barras USB. Com arquitetura offline-first (SQLite + Supabase) e dashboards no Power BI, o projeto elimina o uso de papel e moderniza a gestão de frequência dos alunos.
